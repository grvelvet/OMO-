# Omoglyph Pro

Профессиональный инструмент обфускации текста с использованием омоглифов и алгоритмов разрыва сигнатур автоматизированных систем и ИИ.

## О проекте
Omoglyph Pro защищает ваш контент от автоматического парсинга, анализаторов текста и систем распознавания. Приложение заменяет стандартные символы на визуально идентичные символы из других алфавитов (омоглифы) и использует невидимые разделители (Zero-width spaces) для искусственного разрыва словарных токенов ИИ без изменения визуального отображения текста для человека.

### Основные возможности
- **Интеллектуальная замена символов**: Умная замена букв на омоглифы с настраиваемой интенсивностью применения.
- **Разрыв токенов**: Вставка невидимых символов (ZWS) для алгоритмического запутывания систем.
- **Полная приватность**: Вся обработка текста происходит строго локально в браузере, ничего не отправляется на сторонние серверы.
- **Просмотр изменений**: Наглядная подсветка модифицированных участков текста.

## Стек технологий
- React 19
- TypeScript
- Vite

## Запуск для разработки

1. Склонируйте репозиторий:
   ```bash
   git clone <url-вашего-репозитория>
   cd <папка-проекта>
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сервер для разработки локально:
   ```bash
   npm run dev
   ```

## Развертывание (Деплой)
Проект изначально оптимизирован и настроен для автоматической сборки и публикации на **GitHub Pages**.

В репозитории уже присутствует файл автоматизации (GitHub Actions) по пути `.github/workflows/deploy.yml`. Достаточно просто запушить код в ветку `main`, и GitHub автоматически соберет проект и развернет его.
